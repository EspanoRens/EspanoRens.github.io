Reflection & Learnings

Reflections

This course was all about teaching us on how to solve problems that are usually encountered by administrators who manage enterprise servers. Most of the activities are created to simulate tasks that are commonly faced by administrators. Because of this, I also get to experience the problems that they face like unexpected errors when running programs, or most commonly in installing programs.Those instances really taught me how to read the errors and how to look up how to fix the errors. I also learned how to manage and control different machines remotely with the use of SSH to go into the servers and use Ansible to automatically install large files to multiple servers without going to each server to install them manually. I also experienced alot of problems with using Ansible since it was new to me, but I overcame and learned from the errors so the process became easier and overall smoother to do. Overall, this course gave me an understanding on how Linux distribution is used at an enterprise level and what are the many problems that administrators face. I did experience problems but I was able to overcome them and learn from them and I will use these experiences for my future career.

Learnings

This course has taught me alot about the concepts of managing enterprise servers with the DevOps culture. I was able to gain an understanding on how diverse the applications of the Linux distribution and how important it is to the workforce. With the use of Linux I also learned how to set up SSH and how to use it to communicate and control servers and use Ansible to remotely control other machines and install different programs into them. Overall, the course taught me basic to intermediate skills that I will use for my future career.
